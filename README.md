# Food Delivery Application (Java Swing)

## 📌 Overview

This project is a simple Food Delivery Application built using Java Swing. It allows users to select food items and choose a payment method to place an order.

---

## 🚀 Features

* GUI-based application using Java Swing
* Uses Strategy Design Pattern
* Multiple payment options (Cash, Card, UPI)
* Dynamic order processing
* Multithreading for delay simulation
* Simple and user-friendly interface

---

## 🛠️ Technologies Used

* Java
* Swing (GUI)
* Multithreading
* Object-Oriented Programming (OOP)
* Strategy Design Pattern

---

## 📂 Project Structure

* `Project10.java` → Main class with GUI and logic
* `PaymentStrategy` → Interface
* `CashPayment`, `CardPayment`, `UpiPayment` → Strategy classes
* `Order` → Context class

---

## ▶️ How to Run

1. Open Command Prompt / Terminal
2. Compile the program:

   ```bash
   javac Project10.java
   ```
3. Run the program:

   ```bash
   java Project10
   ```

---

## 📊 How It Works

* User selects a food item (Pizza or Burger)
* Chooses a payment method (Cash, Card, UPI)
* Clicks **Place Order**
* The system processes the order using selected strategy
* Result is displayed after a short delay
